## 🛠️ Tehnologii utilizate

- **React 18** + **Vite** — framework frontend
- **Tailwind CSS** — styling utility-first
- **React Router DOM** — navigare între pagini
- **Strapi** — CMS Headless (backend)
- **Netlify** — hosting frontend
- **GitHub** — versionare cod
